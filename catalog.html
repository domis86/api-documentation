Catalog

The key part of the Musement catalogue is events. Events are organized by type using Categories and Verticals and by geographical location using Cities and Venues.

To find events in the catalogue you can use the Search or you can navigate to it by using some predefined collection.

Event endpoints

All these endpoints return events. All these endpoints support different parameters that allow you to further filter the events.

  • GET /api/v3/verticals/{id}/events | All the events for a specific Vertical | Test it
  • GET /api/v3/categories/{id}/events | All the events for a specific Category | Test it
  • GET /api/v3/venues/{id}/events | All the events for a specific Venue | Test it
  • GET /api/v3/cities/{id}/events | All the events for a specific City | Test it
  • GET /api/v3/cities/{id}/events/{today|tomorrow} | Return all events available in a specific city today or tomorrow | Test it
  • GET /api/v3/countries/{id}/events | All the events for a specific Country | Test it

Full list: http://api.musement.com/documentation
